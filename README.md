📘 Multi-page Student Profile Website

📌 Activity Description

This project is a simple multi-page website that demonstrates basic HTML5 structure, navigation, forms, images, and lists. It is designed as a laboratory activity to strengthen foundational web development skills.

🧩 Features

    - Navigation between pages
    - Profile image
    - List of hobbies
    - Simple contact form

📁 Project Structure

    student-profile/
      │── index.html (Bio Page)
      │── contact.html (Contact Form Page)
      │── images/
        │ └── profile.jpg

🧠 What I learned

    - HTML structure
    - Links between pages
    - Using images
    - Creating forms
    - Using lists
